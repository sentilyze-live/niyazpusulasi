# Niyaz Pusulası — Abonelik ve Fiyatlandırma Stratejisi

## Temel Felsefe

**"Dini içerik ücretsiz, deneyim premium"**

Namaz vakitleri, kıble, temel bildirimler gibi İslami temel işlevler HER ZAMAN ücretsiz kalacak.
Premium abonelik = daha güzel, daha zengin, daha kişiselleştirilmiş deneyim.
Hard paywall YOK — kullanıcı premium özelliğe dokunduğunda contextual soft paywall gösterilir.

## Soft Paywall Stratejisi

### Paywall Gösterilme Anları (Contextual Triggers)

Paywall ASLA ana ekranda veya uygulama açılışında gösterilmez. Kullanıcı premium bir özelliğe dokunduğunda gösterilir:

| Tetikleyici | Kullanıcı Aksiyonu | Paywall Mesajı |
|-------------|-------------------|----------------|
| 4. alışkanlık ekleme | "Yeni Alışkanlık" butonuna tıklama | "Sınırsız alışkanlık ile hedeflerine ulaş" |
| Ezan sesi değiştirme | Ses ayarlarında premium ses seçme | "10+ müezzin sesi ile namazına hazırlan" |
| Lock screen widget | Widget ayarlarında kilit ekranı widget seçme | "Kilit ekranında namaz vakitlerini gör" |
| Premium tema | Tema ayarlarında premium tema seçme | "Kişiselleştirilmiş tema ile uygulamanı güzelleştir" |
| İstatistik görme | Alışkanlık grafiğine tıklama | "Detaylı istatistiklerle gelişimini takip et" |
| Gelişmiş bildirim | Özel offset seçme veya çoklu hatırlatma | "Bildirimleri tam istediğin gibi ayarla" |
| Tesbih sayacı | Tesbih özelliğine erişme | "Dijital tesbih ile zikrini kolaylaştır" |

## Dönüşüm Optimizasyonu

### Beklenen Metrikler

| Metrik | Hedef | Sektör Ortalaması |
|--------|-------|-------------------|
| İndirme → Deneme Başlatma | %5-8 | %4-6 (freemium) |
| Deneme → Ödeme | %35-45 | %30-45 (düşük fiyat) |
| İndirme → Ödeme (toplam) | %2-4 | %2,18 (freemium median) |
| Aylık Churn | %8-12 | %10-15 |
| Yıllık Retention (12 ay) | %40-50 | %36 (düşük fiyat median) |
| Yıllık/Aylık Dağılımı | %65/%35 | %60/%40 |

### Dönüşümü Artıran Taktikler

1. **Ramazan Kampanyası** (Kritik Dönem)
   - Ramazan öncesi 1 hafta: Yıllık plan %30 indirimli (149,99 → 104,99 TL)
   - "Ramazan'a Hazır mısın?" kampanya mesajı
   - Ramazan boyunca premium İftar/İmsak widget'ı gösterimi
   - Ramazan ayı en yüksek dönüşüm potansiyeli

2. **Onboarding Value Demonstration**
   - İlk 3 gün: Premium ezan seslerinden birini ücretsiz deneme
   - İlk hafta: Tüm widget'ları ücretsiz göster, sonra soft lock
   - 7. gün: "Premium deneyimini beğendin mi?" push notification

3. **Social Proof**
   - Paywall'da: "X kişi bu ay premium'a geçti"
   - App Store açıklamasında: Kullanıcı yorumları vurgusu

4. **Streak Koruma**
   - 7+ gün streak olan kullanıcıya: "Streak'ini premium istatistiklerle analiz et"
   - Kaybedilen streak sonrası: "Premium ile streak hatırlatıcıları al"

| Metrik | Değer |
|--------|-------|
| Aylık indirme | 10.000 |

**Aylık Gelir Hesabı (6. ay itibarıyla, churn dahil):**

| Kalem | Hesap | Gelir |
|-------|-------|-------|
| Aktif yıllık aboneler | ~900 (6 ay birikim, %5 churn/ay) | 900 × 12,50 TL/ay = 11.250 TL |
| Aktif aylık aboneler | ~350 (dönen havuz, %12 churn/ay) | 350 × 29,99 TL = 10.497 TL |
| **Toplam Brüt** | | **~21.747 TL/ay** |
| Apple komisyonu (%30, ilk yıl %15) | | -3.262 TL |
| **Net Gelir** | | **~18.485 TL/ay** (~$422) |

**12. ay itibarıyla:**
- Aktif abone tabanı: ~2.000-2.500
- Aylık brüt gelir: ~45.000-55.000 TL (~$1.000-1.250)
- Yıllık toplam brüt: ~350.000-450.000 TL (~$8.000-10.000)

> Not: Apple Small Business Program ile ilk $1M gelire kadar komisyon %15 (standart %30 yerine).

### Entitlement Kontrolü

```swift
// Premium erişim kontrolü
func isPremium() async -> Bool {
    let customerInfo = try? await Purchases.shared.customerInfo()
    return customerInfo?.entitlements["premium"]?.isActive == true
}

// Paywall gösterimi
func showPaywallIfNeeded(for feature: PremiumFeature) {
    Task {
        guard !(await isPremium()) else { return }
        showPaywall = true
        paywallTrigger = feature
    }
}
```

## Özet: Neden Bu Fiyatlandırma?

1. **29,99 TL/ay** — Apple Music'in yarısı fiyatına, günde 1 TL. Türk kullanıcı için ulaşılabilir.
2. **149,99 TL/yıl** — %58 tasarruf ile yıllık plana yönlendirme. Yıllık plan = düşük churn + öngörülebilir gelir.
3. **$3,99/ay global** — Muslim Pro'dan ucuz, Athan Pro'nun yarısı. Agresif ama sürdürülebilir.
4. **Hard paywall yok** — Dini uygulama kullanıcıları temel işlevlerin ücretsiz olmasını bekler. Paywall sadece premium deneyim için.
5. **7 gün deneme** — Düşük giriş bariyeri, yüksek deneme-ödeme dönüşümü (%35-45).
6. **Ramazan odaklı büyüme** — Yılın en kritik dönemi, indirmeler 3-5x, dönüşüm 2x.
